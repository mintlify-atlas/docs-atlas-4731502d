## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** 18 or later
- A package manager: **npm** (bundled with Node.js), **yarn**, or **pnpm**
- **Git**

## Set up the project

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/ajipitchayapong/projectSecret.git
    cd projectSecret
    ```
  </Step>

  <Step title="Install dependencies">
    <CodeGroup>
      ```bash npm
      npm install
      ```
      ```bash yarn
      yarn install
      ```
      ```bash pnpm
      pnpm install
      ```
    </CodeGroup>

    This installs all runtime and dev dependencies defined in `package.json`, including React 19, Framer Motion, Three.js, and React Router.
  </Step>

  <Step title="Start the development server">
    <CodeGroup>
      ```bash npm
      npm run dev
      ```
      ```bash yarn
      yarn dev
      ```
      ```bash pnpm
      pnpm dev
      ```
    </CodeGroup>

    Vite starts the dev server with hot module replacement (HMR). By default it listens on all interfaces (`host: true`) so you can also access it from other devices on the same network.

    ```
    VITE v7.x.x  ready in XXXms

    ➜  Local:   http://localhost:5173/
    ➜  Network: http://192.168.x.x:5173/
    ```
  </Step>

  <Step title="Open the app">
    Navigate to [http://localhost:5173](http://localhost:5173) in your browser.

    The root path `/` redirects to `/stages`, where the three-stage navigation hub loads with an animated intro sequence.
  </Step>
</Steps>

## Available routes

| Route | Page | Description |
|---|---|---|
| `/` | — | Redirects to `/stages` |
| `/stages` | Stages hub | Animated intro + three-stage navigation |
| `/landing` | Ocean explorer | Parallax scroll scene with all 10 animals |
| `/detail` | Animal detail | Polygon shard animation + species info |
| `/detail?id=<animalId>` | Animal detail | Opens directly to a specific animal |
| `/polygon` | Why Polygon? | Horizontal scroll explainer scene |
| `/order` | Orderer (dev tool) | Internal tool for configuring shard animation order |

## Build for production

<CodeGroup>
  ```bash npm
  npm run build
  ```
  ```bash yarn
  yarn build
  ```
  ```bash pnpm
  pnpm build
  ```
</CodeGroup>

Vite bundles the app into the `dist/` directory. To preview the production build locally:

<CodeGroup>
  ```bash npm
  npm run preview
  ```
  ```bash yarn
  yarn preview
  ```
  ```bash pnpm
  pnpm preview
  ```
</CodeGroup>

<Note>
The production build requires the SPA rewrite rule configured in `vercel.json` (or an equivalent server-side rewrite) so that all routes serve `index.html`. See the [Deployment guide](/development/deployment) for details.
</Note>
